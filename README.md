# Python Assignment 2

## Overview
This repository contains two Python programs created as part of Assignment 2.  
These programs focus on conditional statements, loops, and basic logic building in Python.

---

## Task 1: Even or Odd Checker
This program checks whether a given number is even or odd.

### Functionality
- Takes an integer input from the user
- Uses an if-else statement to check whether the number is even or odd
- Displays the result accordingly

### Concepts Used
- User input
- Integer data type
- If-else condition
- Modulus operator

---

## Task 2: Sum of Numbers from 1 to 50
This program calculates the sum of numbers within a fixed range.

### Functionality
- Uses a for loop to iterate over numbers from 1 to 50
- Calculates the sum of all integers in this range
- Displays the final sum

### Concepts Used
- For loop
- Variables
- Arithmetic operations
- Print statement
